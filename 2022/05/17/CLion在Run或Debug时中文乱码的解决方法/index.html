CLion在Run或Debug时中文乱码的解决

Clion 默认全局编码方式为 UTF-8，会导致中文乱码

官方 JetBrains 提供的解决方案是：
使用快捷键：shift+ctrl+alt+/ 然后选择第一个选项 Registry（注册表），找到蓝色的 run.processes.with.pty 将其 Value 的复选框去掉，即可正常显示中文。
